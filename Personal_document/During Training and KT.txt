5. Command for SQL server
	#DDL(data definition language):- Create, Alter, Drop, Truncate, Rename (sp_rename 'schema.db_name', 'new_name', 'column')
	#DML(data manipulated language):- Insert, Update ({Update [db_name] set [column_name] = [input data];}), Delete
	#TCL(Transaction control language):- Commit, Rollback (always used being transaction)
	#DCL(data control language):- Grant, Revoke
	#DQL(data query language):- Select
